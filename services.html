Services

Web Development

I design and build small, reliable websites and dashboards that load quickly and feel simple to navigate. I focus on accessible markup, clean layouts, and the details that make a site pleasant—spacing, contrast, keyboard navigation, and sensible defaults. For projects that need more behavior, I add lightweight JavaScript without over‑engineering.

Automation & Scripting

Busywork is expensive. I write Python or PowerShell scripts to automate recurring tasks like data cleanup, CSV/Excel workflows, and report generation. I document assumptions, add clear logs, and make the scripts easy for non‑developers to run.

IT & Systems Support

I’m comfortable helping set up Windows/M365 devices, organizing files and permissions, improving Wi‑Fi reliability, and writing straightforward documentation so people can help themselves. I care about patient communication and leaving systems a little better than I found them.

Data & Analysis

For datasets that need cleaning and simple modeling, I use Python (Pandas) and SQL to explore, transform, and visualize results. I focus on clarity and reproducibility so others can follow the steps and trust the conclusions.
